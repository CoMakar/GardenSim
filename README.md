# GardenSim

~ A project to model an abstract plant growing process.

```shell
# enter the shell
$ pipenv shell

# ----------------------------
# install dependencies
$ pipenv install

# run
$ python main.py
# ----------------------------

# | or |

# ----------------------------
# install dev dependencies
$ pipenv install --dev

# build
$ pyinstaller main.spec
# ----------------------------
```
